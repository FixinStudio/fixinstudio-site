Privacy.

Short version: this website collects nothing about you.

Last updated: July 2, 2026.

This policy covers fixinstudio.com, the website of Fixin Studio LLC. Our products each have their own privacy policy; see the product’s own page or App Store listing.

What we collect on this site

Nothing. This is a static website with no accounts, no forms, no cookies, and no analytics or advertising trackers. We don’t build a profile of you, and we have no data about you to sell — and we wouldn’t anyway.

Hosting

The site is hosted on GitHub Pages. Like any web host, GitHub may automatically record standard technical request data (such as IP addresses) in its server logs for security and operation, as described in GitHub’s privacy statement. We don’t receive or use that data.

Fonts

This site loads the Inter and Fraunces typefaces from Google Fonts. Google may log request metadata (IP address, browser) as described in Google’s privacy policy. No other third-party resources are loaded.

If you email us

If you contact us at hello@fixinstudio.com, we’ll use your message and email address only to reply and to help you. We don’t add you to any list.

Changes

If this policy changes, we’ll update the date above. Questions? Email hello@fixinstudio.com.
